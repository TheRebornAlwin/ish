## Customization

### Colors
Edit CSS custom properties in `<style>` tag:
```css
:root {
    --cream: #F5F3EF;
    --charcoal: #2C2C2C;
    --warm-gold: #C8A882;
    --warm-beige: #E8E3DA;
}
```

### Images
Replace image URLs in the HTML with your own. Current images from:
- IMAGE_URLS_REFERENCE.txt (branding/services)
- Unsplash (content placeholders)

### Content
All text is easily editable in the HTML files.
